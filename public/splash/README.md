## Design Guidelines

- Use the Ongea Pesa green gradient background (#22c55e to #16a34a)
- Center the app logo/icon
- Add app name below the icon
- Keep design minimal and clean

## Generation Tools

Use [PWA Asset Generator](https://www.pwabuilder.com/imageGenerator) or [Progressier](https://progressier.com/pwa-splash-screen-generator) to generate all splash screens from a single source image.
