## üéØ Introduction to Trademarks

### What is a Trademark?

A **trademark** is a distinctive sign, symbol, word, phrase, logo, or combination thereof that identifies and distinguishes the source of goods or services of one party from those of others.

**Legal Definition (Kenya Trade Marks Act, Cap. 506):**
> "A mark includes a device, brand, heading, label, ticket, name, signature, word, letter, numeral, or any combination thereof."

### Types of Trademarks

| Type | Description | Example for Ongea Pesa |
|------|-------------|------------------------|
| **Word Mark** | Text-only trademark | "Ongea Pesa" |
| **Logo/Device Mark** | Graphical symbol or design | Ongea Pesa logo |
| **Combined Mark** | Word + logo together | "Ongea Pesa" + logo |
| **Slogan/Tagline** | Distinctive phrase | "Voice Your Payment" |
| **Service Mark** | Identifies services (not goods) | Financial services branding |
| **Collective Mark** | Used by association members | Industry association mark |
| **Certification Mark** | Certifies quality/origin | Security certification |

### What Can Be Registered?

‚úÖ **Registrable:**
- Distinctive words (invented, arbitrary, suggestive)
- Logos and symbols
- Unique color combinations
- Shapes (if distinctive)
- Sounds (in some jurisdictions)
- Slogans (if distinctive)

‚ùå **Not Registrable:**
- Generic terms ("Payment App")
- Descriptive terms ("Fast Money Transfer")
- Deceptive marks
- Offensive/immoral marks
- Government symbols/flags
- Identical/confusingly similar to existing marks

## üá∞üá™ Kenya Trademark Registration Process

### Overview

**Governing Law:** Trade Marks Act (Cap. 506)  
**Regulatory Body:** Kenya Industrial Property Institute (KIPI)  
**System:** First-to-file (not first-to-use)  
**Duration:** 10 years, renewable indefinitely

### Step-by-Step Process

#### **Step 1: Preliminary Trademark Search (TM27)**

**Purpose:** Check if your proposed mark conflicts with existing registrations

**Process:**
1. Visit KIPI eCitizen portal: [https://kipi.ecitizen.go.ke](https://kipi.ecitizen.go.ke)
2. Select "Trademark Search (TM27)"
3. Enter proposed mark details
4. Select relevant Nice class(es)
5. Pay search fee
6. Receive search report (3-5 business days)

**Cost:**
- Local applicant: KES 1,000 per class
- Foreign applicant: USD 50 per class

**What to Look For:**
- Identical marks in same class
- Similar marks in same/related classes
- Phonetically similar marks
- Visually similar marks
- Conceptually similar marks

**‚ö†Ô∏è Important:** A clear search doesn't guarantee registration, but reduces rejection risk.

---

#### **Step 2: Choose Nice Classification**

Trademarks are registered per class of goods/services. Kenya uses the **Nice Classification** system (45 classes total).

**Recommended Classes for Ongea Pesa:**

| Class | Description | Ongea Pesa Application |
|-------|-------------|------------------------|
| **Class 9** | Computer software, mobile applications, downloadable software | ‚úÖ **Critical** - Mobile app, software platform |
| **Class 36** | Financial services, payment processing, electronic funds transfer, mobile banking | ‚úÖ **Critical** - Core payment services |
| **Class 38** | Telecommunications, data transmission, electronic messaging | ‚úÖ **High Priority** - Voice communication, data services |
| **Class 42** | Software as a service (SaaS), cloud computing, software development | ‚úÖ **High Priority** - Platform services |
| **Class 35** | Advertising, business management, data processing | üü° Medium Priority - Marketing services |
| **Class 41** | Education, training services | üü¢ Low Priority - User training |

**Strategy:**
- **Phase 1:** Register Classes 9, 36, 38 (core business)
- **Phase 2:** Add Class 42 (SaaS expansion)
- **Phase 3:** Add Classes 35, 41 (ancillary services)

**Cost Consideration:** Each class requires separate filing fee

---

#### **Step 3: Prepare Application Documents**

**Required Information:**

1. **Applicant Details:**
   - Full legal name (as per Certificate of Incorporation)
   - Registered office address in Kenya
   - Postal address
   - Email and phone contact
   - Nationality/country of incorporation
   - Company registration number

2. **Mark Representation:**
   - **Word Mark:** Exact spelling and capitalization
   - **Logo Mark:** High-resolution image (JPEG/PNG, min 300 DPI)
   - **Combined Mark:** Logo + text as single image
   - **Color Claims:** Specify if claiming specific colors

3. **Goods/Services Specification:**
   - Precise description within chosen class(es)
   - Use standard Nice Classification terminology
   - Avoid overly broad or vague descriptions

**Example Specification for Class 36:**
```
Financial services; payment processing services; electronic funds transfer; 
mobile payment services; digital wallet services; money transfer services; 
bill payment services; financial transaction services via mobile devices; 
voice-activated payment services.
```

4. **Additional Documents (if applicable):**
   - **Foreign Applicants:** TM1 (Appointment of Local Agent)
   - **Priority Claim:** If claiming priority from earlier foreign application
   - **Proof of Use:** If claiming prior use (optional but helpful)

---

#### **Step 4: File TM2 Application**

**Filing Methods:**

1. **Online (Recommended):**
   - Portal: [https://kipi.ecitizen.go.ke](https://kipi.ecitizen.go.ke)
   - Create account or log in
   - Complete TM2 form online
   - Upload mark representation
   - Pay via M-Pesa, card, or bank
   - Receive application number immediately

2. **Physical Filing:**
   - Visit KIPI offices (Kapiti Road, Nairobi)
   - Submit completed TM2 form
   - Pay at cashier or bank
   - Receive stamped acknowledgment

**Filing Fees (Per Class):**

| Applicant Type | First Class | Additional Classes |
|----------------|-------------|-------------------|
| **Local (Kenyan resident/business)** | KES 4,000 | KES 3,000 each |
| **Foreign (non-resident)** | USD 150 | USD 100 each |

**Payment Methods:**
- M-Pesa (Paybill: 222111, Account: Application Number)
- Credit/Debit Card
- Bank deposit (Kenya Commercial Bank)

**Output:** Application number (e.g., TM/2026/12345)

---

#### **Step 5: Examination by KIPI**

**Timeline:** 3-6 months after filing

**What KIPI Examines:**

1. **Formalities:**
   - Correct forms and fees paid
   - Proper applicant details
   - Clear mark representation

2. **Absolute Grounds for Refusal:**
   - Mark is not distinctive
   - Mark is descriptive/generic
   - Mark is deceptive or offensive
   - Mark violates public policy

3. **Relative Grounds for Refusal:**
   - Conflicts with earlier registered/pending marks
   - Confusingly similar to existing marks
   - Bad faith application

**Possible Outcomes:**

‚úÖ **Accepted:** Application proceeds to publication  
‚ö†Ô∏è **Objection Raised:** Examiner issues report with concerns  
‚ùå **Rejected:** Application refused (can appeal)

**If Objection Raised:**
- You receive **Examination Report** via email/portal
- **Response Deadline:** 60 days from report date
- **Options:**
  - Amend specification
  - Provide arguments/evidence
  - Disclaim descriptive elements
  - Withdraw application

**Tips for Responding:**
- Address each objection point-by-point
- Provide evidence of distinctiveness (if claiming acquired distinctiveness)
- Cite favorable precedents
- Consider engaging trademark attorney for complex objections

---

#### **Step 6: Publication in IP Journal**

**Timeline:** 1-2 months after acceptance

**Process:**
- KIPI publishes accepted applications in **Kenya Industrial Property Journal**
- Journal published monthly on KIPI website
- Publication includes: mark, applicant, class, goods/services

**Purpose:** Allows third parties to oppose if they have prior rights

**Opposition Period:** 60 days from publication date

---

#### **Step 7: Opposition Period**

**Who Can Oppose:**
- Owners of earlier similar marks
- Parties with prior rights (e.g., company name, domain)
- Anyone with legitimate interest

**Grounds for Opposition:**
- Confusing similarity to opponent's mark
- Bad faith application
- Mark is descriptive/generic
- Applicant has no right to mark

**If Opposition Filed:**
1. You receive **Notice of Opposition**
2. **Response Deadline:** 60 days
3. File **Counter-Statement** defending application
4. Evidence rounds (both parties)
5. Hearing before KIPI (if requested)
6. Decision by Registrar

**Timeline:** 6-18 months (if opposed)

**Cost:** Legal fees for defense (KES 50,000 - 200,000+)

**‚ö†Ô∏è Important:** Most applications are NOT opposed. Opposition rate in Kenya is ~5-10%.

---

#### **Step 8: Registration**

**If No Opposition (or Opposition Defeated):**

1. KIPI issues **Certificate of Registration**
2. Mark registered for 10 years from filing date
3. You can now use ¬Æ symbol
4. Registration published in IP Journal

**Timeline from Filing to Registration:**
- **Smooth Process:** 8-12 months
- **With Objections:** 12-18 months
- **With Opposition:** 18-24+ months

**What You Receive:**
- Official Certificate of Registration
- Registration number
- Registered mark representation
- Class and specification
- Validity period (10 years)

### Why International Protection?

**Key Principle:** Trademarks are **territorial** - Kenyan registration only protects in Kenya.

### International Filing Options

#### **Option 1: Madrid Protocol (Recommended for Multiple Countries)**

**What is Madrid Protocol?**
- International treaty administered by WIPO
- File ONE application to protect in 130+ countries
- Manage all registrations through single system

**How It Works:**
1. Must have **base application/registration** in Kenya (home country)
2. File **International Application (MM2)** via KIPI
3. Designate countries where you want protection
4. WIPO examines and forwards to designated countries
5. Each country examines under local law
6. If approved, protection equivalent to national registration

**Advantages:**
- ‚úÖ Cost-effective for multiple countries
- ‚úÖ Single application in English
- ‚úÖ Centralized management
- ‚úÖ Easy to add countries later
- ‚úÖ Renewal through single process

**Disadvantages:**
- ‚ùå Dependent on base application for 5 years ("central attack")
- ‚ùå Not all countries are members
- ‚ùå Each country can still refuse

**Cost:**
- **Basic Fee:** CHF 653 (word mark) or CHF 903 (logo)
- **Per-Country Fee:** CHF 100-300 per country (varies)
- **KIPI Handling Fee:** KES 5,000

**Example Cost for 5 African Countries:**
- Basic fee: ~USD 700
- 5 countries √ó USD 150: ~USD 750
- **Total: ~USD 1,450**

**Timeline:** 12-18 months per country

**ARIPO Member States (20 countries):**
Botswana, Cape Verde, Eswatini, Gambia, Ghana, Kenya, Lesotho, Liberia, Malawi, Mauritius, Mozambique, Namibia, Rwanda, S√£o Tom√© and Pr√≠ncipe, Seychelles, Sierra Leone, Somalia, Sudan, Tanzania, Uganda, Zambia, Zimbabwe

## ‚öñÔ∏è Opposition and Enforcement

### Defending Against Opposition

**If Someone Opposes Your Application:**

**Step 1: Receive Notice**
- KIPI sends Notice of Opposition
- Includes opponent's grounds
- 60-day deadline to respond

**Step 2: Assess Grounds**
- Review opponent's claims
- Check their trademark registration
- Assess likelihood of success

**Step 3: Decide Strategy**
- **Fight:** File Counter-Statement
- **Negotiate:** Settle with opponent (co-existence agreement)
- **Withdraw:** Abandon application (if weak case)

**Step 4: File Counter-Statement**
- Address each ground of opposition
- Provide evidence of right to mark
- Argue distinctiveness/no confusion
- Deadline: 60 days

**Step 5: Evidence Phase**
- Both parties file evidence
- Declarations, exhibits, expert opinions
- Timeline: 3-6 months

**Step 6: Hearing (Optional)**
- Either party can request hearing
- Present arguments before Registrar
- Cross-examination of witnesses

**Step 7: Decision**
- Registrar issues written decision
- Application allowed or refused
- Can appeal to High Court

**Cost:** KES 50,000 - 200,000+ (legal fees)

---

### Opposing Others' Applications

**When to Oppose:**
- Competitor files confusingly similar mark
- Someone copies your brand
- Application threatens your business

**Process:**
1. Monitor IP Journal publications
2. File Notice of Opposition (TM5) within 60 days
3. Pay opposition fee: KES 5,000 (local) / USD 200 (foreign)
4. State grounds clearly
5. Provide evidence of prior rights

**Grounds:**
- Confusing similarity
- Your prior use/registration
- Bad faith
- Descriptive/generic mark

---

### Enforcing Your Trademark Rights

#### **Infringement Actions**

**What is Infringement?**
- Unauthorized use of identical/similar mark
- Use on same/similar goods/services
- Likely to cause confusion

**Types of Infringement:**
1. **Identical mark, identical goods** (clear infringement)
2. **Similar mark, similar goods** (likelihood of confusion)
3. **Dilution** (famous marks only)
4. **Passing off** (even without registration)

**Enforcement Options:**

**1. Cease and Desist Letter**
- Send formal demand to stop use
- Give deadline (14-30 days)
- Threaten legal action
- Cost: KES 10,000 - 50,000 (legal fees)

**2. Negotiated Settlement**
- Discuss co-existence
- License agreement
- Rebrand agreement
- Cost: Negotiable

**3. Court Action (High Court of Kenya)**
- File trademark infringement suit
- Seek injunction + damages
- Timeline: 1-3 years
- Cost: KES 200,000 - 1,000,000+

**Remedies Available:**
- ‚úÖ Injunction (stop infringement)
- ‚úÖ Damages (compensation)
- ‚úÖ Account of profits
- ‚úÖ Delivery up/destruction of infringing goods
- ‚úÖ Legal costs

## ‚ùì FAQs

### General Questions

**Q: How long does trademark registration take in Kenya?**  
A: 8-18 months for straightforward cases; 18-24+ months if opposed.

**Q: Can I use my trademark before registration?**  
A: Yes, but you have weaker legal rights. Use ‚Ñ¢ symbol, not ¬Æ.

**Q: What's the difference between ‚Ñ¢ and ¬Æ?**  
A: ‚Ñ¢ = unregistered trademark (common law rights)  
¬Æ = registered trademark (statutory rights)  
Use ¬Æ only after registration certificate issued.

**Q: Do I need to register in every country?**  
A: Yes, trademarks are territorial. Kenya registration only protects in Kenya.

**Q: Can foreigners register trademarks in Kenya?**  
A: Yes, but must appoint local agent (TM1 form) and pay higher fees.

**Q: How much does it cost to register a trademark in Kenya?**  
A: KES 4,000-7,000 per class (local); USD 150-250 per class (foreign).

**Q: How long does trademark protection last?**  
A: 10 years, renewable indefinitely every 10 years.

**Q: Can I register a logo and name separately?**  
A: Yes, recommended. File separate applications for word mark and logo.

**Q: What if someone opposes my application?**  
A: You have 60 days to file Counter-Statement. Consider engaging attorney.

**Q: Can I change my trademark after registration?**  
A: No. Any material change requires new application. Minor changes (color) may be allowed.

### Ongea Pesa Specific

**Q: Which classes should Ongea Pesa register?**  
A: Priority: Classes 9 (software), 36 (financial services), 38 (telecommunications).

**Q: Should we register "Ongea Pesa" and logo separately?**  
A: Yes. File separate applications for maximum protection.

**Q: When should we file internationally?**  
A: After Kenya registration obtained (for Madrid Protocol dependency).

**Q: What if a competitor is using similar name?**  
A: Conduct search, send cease and desist, consider opposition if they file application.

**Q: Can we trademark "Voice Payment" or "AI Payment"?**  
A: Unlikely - too descriptive. Consider distinctive slogans instead.

**Q: Should we register in Swahili and English versions?**  
A: If using both versions commercially, yes. File separate applications.

**Q: What about domain names?**  
A: Trademark registration helps enforce domain disputes (UDRP). Register key domains separately.

**Q: Can we license our trademark?**  
A: Yes, after registration. Record license with KIPI (TM24 form).
