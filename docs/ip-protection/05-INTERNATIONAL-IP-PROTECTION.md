### Key Principle: Territoriality

**âš ï¸ Critical Concept:**

**Example:**
- Kenya trademark registration = Protection in Kenya only
- To protect in Tanzania = Must file in Tanzania
- To protect in 50 countries = Must file in 50 countries (or use international systems)

5. WIPO issues international registration number
6. WIPO forwards to designated countries
7. Monitor country-by-country examination

## â“ FAQs

**Q: Do I need to file in every country separately?**  
A: No. Use Madrid Protocol (trademarks), ARIPO (patents), or PCT (patents) for multiple countries with single application.

**Q: How much does international IP protection cost?**  
A: USD 3,000-13,000 for African coverage; USD 30,000-50,000 for global coverage.

**Q: Can I file Madrid application before Kenya registration?**  
A: Yes, but must have Kenya application filed. Registration not required at filing.

**Q: What if my Kenya trademark is refused?**  
A: Madrid application may be affected if refused within 5 years ("central attack"). Consider transformation to national applications.

**Q: Is copyright automatic internationally?**  
A: Yes, via Berne Convention (180+ countries). But registration in Kenya strengthens enforcement.

**Q: Should I use ARIPO or PCT for patents?**  
A: ARIPO for African markets (cheaper). PCT for global markets (wider coverage).

**Q: How long does international trademark registration take?**  
A: 12-18 months per country via Madrid Protocol.

**Q: Can I add countries later?**  
A: Yes. Madrid allows "subsequent designations" anytime.

**Q: What if I can't afford international protection?**  
A: Prioritize key markets (EAC for Ongea Pesa). File in other countries as revenue grows.

**Q: Do I need local agents in each country?**  
A: For Madrid/ARIPO: No (WIPO/ARIPO handles). For direct filings: Yes.
